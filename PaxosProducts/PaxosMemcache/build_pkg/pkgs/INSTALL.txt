インストールの手順を以下に記します。

1) PAXOS memcache 実行用のユーザを作成する。

  # adduser paxos

2) アーカイブファイルを展開する。

   1) で作成したユーザのホームディレクトリにアーカイブファイルを展開してください。
   展開を行う場合、1)で作成したユーザ権限で実行してください。

  $ tar -xzf Ubuntu_14.tar.gz -C $HOME

3) 環境設定スクリプトの実行する。

  $ cd ~/misc
  $ bash ./setup.sh





PAXOS memcache, CSS を実行する全てのマシンで、上記の 1)～3) の手順を行ってください。
